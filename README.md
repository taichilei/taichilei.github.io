# Personal Homepage

This is my personal homepage built with Jekyll, hosted on GitHub Pages.

## Features

- ✨ **Minimal Design**: Clean and elegant interface, focused on content
- 🌓 **Theme Switcher**: Support light/dark theme toggle, with local storage
- 📱 **Responsive**: Perfectly adapted to desktop and mobile devices
- 🚀 **Fast**: Static site generated by Jekyll, extremely fast loading speed
- 🔧 **Easy to Maintain**: Content written in Markdown, easy to update

## Tech Stack

- **Jekyll**: Static site generator, natively supported by GitHub Pages
- **HTML5/CSS3**: Modern web technologies
- **Vanilla JavaScript**: No heavy frameworks, lightweight and fast

## Local Development

### Option 1: Using Jekyll (Recommended)

```bash
# Install dependencies
bundle install

# Start local server
bundle exec jekyll serve
```

Visit `http://localhost:4000` to see the site.

### Option 2: Simple HTTP Server

```bash
# Use Python to start a simple server
python3 -m http.server 8080
```

Visit `http://localhost:8080/preview.html` to see a preview.

## Deployment

This site is automatically deployed by GitHub Pages. Just push changes to the `main` branch, and GitHub will build and deploy the site automatically.

**Deployment Steps:**

1. Push all files to your GitHub repository named `yourusername.github.io`
2. Go to repository Settings → Pages
3. Set source to `main` branch and root directory
4. Wait a few minutes, your site will be live at `https://yourusername.github.io`

## Customization

- Update personal information in `_config.yml`
- Modify homepage content in `index.md`
- Adjust navigation links in `_config.yml`
- Change styles in `assets/css/style.css`
